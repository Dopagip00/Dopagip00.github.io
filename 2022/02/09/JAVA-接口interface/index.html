1.接口只有规范，无法写方法，是专业的抽象。实现约束和实现分离，面向接口编程。

2.接口就是规范，约束了编程的行为

写一个接口：

（抽象）类只能单继承，但是接口可以多继承，这就是接口的好处；

继承接口用关键字implement，格式为：class+类名+implement+接口名{}

实现了接口中的类就要重写接口中的方法，接口的多继承用逗号连接实现。
