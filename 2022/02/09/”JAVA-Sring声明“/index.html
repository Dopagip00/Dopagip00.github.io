”JAVA--Sring声明“

引用类型的引用在栈里面，对象在堆里面。

字符串的字面值存储在字符串池中，可以实现共享。

String name = 20;//这里name存储在堆里面，20存储在方法池里面。

声明string字符串有两种方式：

1.String name = “Hellp”//产生一个对象，存储在字符串池中

2.String name = new String(“Hello”)//产生两个对象，在字符串池和堆中各存放一个
